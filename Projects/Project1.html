CarWash App

Aplikasi pemesanan cuci mobil berbasis web & mobile

📌 Deskripsi Proyek

CarWash App adalah aplikasi berbasis web dan mobile yang memudahkan pelanggan dalam memesan layanan cuci mobil secara online. Dengan fitur pemesanan berbasis lokasi, pelanggan bisa memilih tempat cuci mobil terdekat, memilih paket layanan, serta melihat status cuci mobil secara real-time.

🚀 Fitur Utama

🔍 Pemesanan Online

Pelanggan bisa memesan layanan cuci mobil dengan mudah melalui aplikasi.

📍 Lokasi Terdekat

Menampilkan daftar tempat cuci mobil terdekat berdasarkan GPS pengguna.

📊 Tracking Status

Pelanggan bisa melihat status mobil mereka dalam proses pencucian.

🛠 Teknologi yang Digunakan

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Laravel (PHP)
  • Database: MySQL
  • API: Google Maps API (untuk lokasi cuci mobil)
  • Mobile App: Kotlin (Android) & Flutter (Cross-Platform)
