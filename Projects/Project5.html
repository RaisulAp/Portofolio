Human Resources Mobile App

Flutter-based app for attendance, KPI monitoring, and employee leave management

📌 Project Overview

A mobile application developed for internal use by a company’s HR department, using Flutter. The app allows employees to mark attendance, monitor their KPI performance, and request leave. Designed with a mobile-first approach, the app emphasizes user experience and responsiveness. It communicates with backend APIs to sync and retrieve employee data in real-time.

👨‍💼 Key Features

🕒 Attendance

Users can record their check-in and check-out times with timestamp validation and geolocation (optional).

📊 KPI Tracking

Employees can track their performance metrics and receive updates directly through the app.

📝 Leave Requests

Enables users to apply for leave and view approval status, integrated with supervisor-side approval system.

⚠ Challenges Faced

  • UI/UX Design: Creating a clean and intuitive layout that simplifies HR workflows.
  • Data Sync: Ensuring real-time accuracy and consistency with backend data.
  • Scalability: Designing architecture ready to support additional modules in the future.

🛠 Technologies Used

This app was developed using:

  • Flutter
  • Dart
  • Laravel API
  • Figma
  • Android SDK
